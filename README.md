# FillInTheBlanksView
填空题View，自动生成填空题


使用方法：

传入字符串以及字体大小和字体颜色，需要填空的文字用()包裹起来，文字会填充完本行后自动换行，Edittext的宽度会根据需要填空的文字来确定，
Edittext宽度超过本行宽度时会自动换行。
